## Auditoría
- **Def**. Revisión o examen independiente para es determinar responsabilidades en seguridad.
	- Idoneidad de controles / Cumplimiento de políticas / Detectar Brechas / Sugerir cambios
	- Se evalúan registros cronológicos de un sistema (registros de auditoría)
- **Registros a capturar (ISO 27002)**
	- Accesos Autorizados e Intentos de acceso no autorizados
	- Operaciones realizadas usando privilegios
	- Alertas por fallos
	- Cambios e intentos de cambio en configuraciones y controles de seguridad
- Niveles de Registro. SO / Software de soporte / aplicación de usuario

## Auditoria en Linux. SysLog
- `syslog` (usa `syslog.conf)
	- `/dev/log` (registrado con `syslog(3)`)
		- User process 
		- `klogd` (registrado en `proc/kmsg` con `syslog(2)`)
			- Kernel (mensajes con `printk()`)
	- UDP port 514 (TCP/IP network)
		- Process on a remote host
