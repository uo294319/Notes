## Kerberos
- Def.
	- Protocolo de auth para sistemas distribuidos.
	- Estrategia descentralizada para auth clientes ante servidores (y opcionalmente al revés)
- Partes
	- Dominio (realm) es un conjunto de nodos gestionados por mismo servidor kerberos
	- Principal es un usuario o servicio conocido identificado con Dominio + Principal

## Inicio de Sesión Único (Single-Sign-on, SSO)
- **Def**. Acceso a multiples aplicaciones/sistemas/recursos con un solo auth.
	- **Ventajas**. Reduce numero contraseñas / Tiempo introducir / Coste de gestión
	- **Desventajas**. Robo de única contraseña / punto único de fallo
- **Más común**: Inicio de Sesión Web Único (Web-SSO)
	- Ejemplo. CAS (Central Authentication Service)
