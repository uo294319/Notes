## Tipos de Streaming
- Streaming (One-to-Many)
	- Contenido pre-grabado (VOD) o Directo (Live) con buffer
	- Netfilx, Youtube, Twitch
- Tiempo Real / Interactivo (One-to-One/Few)
	- Comunicación bidireccional
	- Latencia Crítica (<200ms)
	- Zoom, Google Meet, MS Teams

## Tipos de Información
### Imágenes 
- **Mapas de bits** (Bitmap o raster)
	- Array de puntos (pixeles)
	- Color representado como n bits (RGB)
	- Se pueden comprimir (redundancia espacial)
		- Sin pérdida (`.bmp`, `.png`, `webp`)
		- Con pérdida (`.jpeg`)
- **Vectorial**
	- Lista de objetos (circulo, recta, arco)
	- Tienen atributos (tamaño, origen, color, grosor)
	- `svg` y otros.
### Audio Digital
- **Representación**
	- Sample Rate (eje temporal)
		- Establece el número de mediciones por segundo
		- Teorema de Nyquist
			- $f_s > 2\times f_{max}$
			- Muestrear al doble de frecuencia máxima para poder reconstruir señal.
		- Estándar: 48 kHz
	- Bit Depth (eje de amplitud)
		- Establece la precisión del sonido (16/24 bit)
		- Estándar: Dinámico (16/24 bit)
- **Codecs**
	- Sin comprimir o *raw* (`PCM` )
	- Compresión sin pérdida o *lossless* (`FLAC`)
	- Codec Modernos (con pérdida)
		- AAC.
			- Estándar para vídeo (`MP4`o `HSL`)
			- Propietario (patentes)
			- Latencia alta
		- Opus
			- Estándar para tiempo real (`WebRTC`)
			- Latencia ultra baja
